## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/cookie-app.git
```

Go to the project directory

```bash
  cd ..cookie-app
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run start
```
